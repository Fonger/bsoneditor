Custom Ace editor

This example demonstrates how to load a custom version of Ace editor into BSONEditor.

By default, BSONEditor code mode loads the following Ace plugins:

  • ace/mode/json
  • ace/ext/searchbox
  • ace/theme/bsoneditor

The bsoneditor theme comes embedded with BSONEditor. The other two plugins (json and searchbox) must be available in the folder of the custom Ace editor, or already be loaded via a script tag.
